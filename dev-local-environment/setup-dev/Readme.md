# Installation

- 1. Install Gvim using the latest version from [here](https://www.vim.org/download.php#pc)
- 2. Copy \_vimrc file to your installation directory

- 3. run ./setup1.sh ( this will install all the plugins and dependencies )
- 4. run PlugInstall in gvim
- 5. quit gvim and open it again ( this will install all coc plugins )
